# Tiny Explorer - AI Development Guide

## Project

Tiny Explorer is a premium mobile application for toddlers (ages 1-4) that helps children discover animals through beautiful illustrations, authentic sounds, and simple interactions.

The goal is to create the highest-quality toddler animal experience, not simply another animal sound app.

## Technology

* React Native
* Expo
* TypeScript
* GitHub
* VS Code

## Development Principles

* Keep the UI simple.
* Optimize for toddlers.
* Optimize for parents' trust.
* Build reusable components.
* Prefer readability over clever code.
* Avoid unnecessary dependencies.
* Keep architecture scalable.
* Use TypeScript.
* Follow React Native best practices.

## Code Style

* Small components.
* Clear naming.
* Self-documenting code.
* Avoid duplication.
* Add comments only when they improve understanding.

## Before Implementing Features

Always:

* Read the documentation inside `/docs`.
* Respect previous architectural decisions.
* Explain major design choices when creating new code.
* Ask for clarification if requirements are ambiguous instead of making large assumptions.

## Goal

Help build a polished, maintainable, production-quality application.
