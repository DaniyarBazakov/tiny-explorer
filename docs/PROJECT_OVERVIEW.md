# Tiny Explorer - Project Overview

## What We Are Building

Tiny Explorer is a mobile app for toddlers ages 1-4. The first version focuses only on animals. Toddlers see friendly animal illustrations, tap them, hear the animal sound, and hear the animal name spoken aloud.

## Product Goal

Build a beautiful, simple, high-quality animal discovery app that toddlers can use without instructions and parents feel good about downloading.

## Target Users

Primary users:

- Toddlers ages 1-4

Secondary users:

- Parents
- Grandparents
- Daycare educators

## Core User Flow

1. Open the app.
2. See animal categories.
3. Tap a category.
4. See large animal cards.
5. Tap an animal.
6. Animal animates.
7. Animal sound plays.
8. Spoken voice says the animal name.
9. Child continues exploring.

## MVP Features

- Home screen
- Animal categories

## Design Direction

The app should feel:

- Calm
- Friendly
- Premium
- Beautiful
- Simple
- Safe
- Magical

The app should not feel:

- Loud
- Cheap
- Cluttered
- Overwhelming
- Confusing

## Art Direction

Use original, AI-assisted illustrations.
Do not use copyrighted characters, brands, studios, artists, or recognizable copyrighted styles.
Animals should feel consistent, friendly, soft, and expressive.
Avoid real animal photos for the main app visuals.

## Sound Direction

Use high-quality animal sounds.
Use a warm, friendly spoken voice for animal names.
Sounds should not be too loud, harsh, or startling.
